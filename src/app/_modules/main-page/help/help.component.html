What is Join?

JOIN is a Kanban Webapplication to structure, time and complete tasks efficiently. By creating accounts for every User and adding contacts to the created tasks, it is easy to share tasks and progression. This application can be used in a Company environment or as solo developer.

How to use it?

1.

In the create task section you are able to create a task. To create a task you have to add a title, category, assigned contacts, finish date, priority, description and if needed subtasks.

2.

After creating a task it is shown in the board section. Every task created in the add task section is automaticly signed as "todo".
By clicking the Add button located at the columns you can create task which are signed as "progress", "feedback" or "done".
Via drag and drop you are able to move the tasks in another state section.
By clicking at the tasks a more detailed dialog is opened. Here you are also able to edit the task.

3.

In the contacts section you can see all your contacts. By clicking the add contact button you are able to add a new contact to the list. You are also able to add a task directly assigned to a contact.
